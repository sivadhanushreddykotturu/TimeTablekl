# Timetable PWA

A Progressive Web App (PWA) for KL University students to access their timetable with ease. This app provides a modern, responsive interface for viewing class schedules, subjects, and academic information.

## 🌟 Features

- **📱 Progressive Web App**: Install on your device like a native app
- **🎨 Modern UI**: Clean, responsive design with dark/light theme support
- **📅 Timetable Management**: View and manage your class schedules
- **📚 Subject Information**: Access detailed subject information
- **🔐 Secure Authentication**: Protected access with login system
- **🌙 Theme Toggle**: Switch between light and dark modes
- **📱 Offline Support**: Works even without internet connection
- **⚡ Fast Performance**: Optimized for quick loading and smooth interactions

## 📱 Installation Instructions

### For Android Users

#### Method 1: Chrome Browser (Recommended)
1. **Open Chrome** on your Android device
2. **Navigate** to the app URL: `https://your-deployment-url.com`
3. **Wait** for the page to load completely
4. **Tap the menu** (three dots) in the top-right corner
5. **Select "Add to Home screen"** or "Install app"
6. **Confirm** the installation when prompted
7. **Launch** the app from your home screen

#### Method 2: Samsung Internet Browser
1. **Open Samsung Internet** browser
2. **Navigate** to the app URL
3. **Tap the menu** (three lines) in the bottom-right
4. **Select "Add page to"** → "Home screen"
5. **Confirm** the installation
6. **Launch** from your home screen

#### Method 3: Firefox Browser
1. **Open Firefox** on your Android device
2. **Navigate** to the app URL

5. **Confirm** the installation

## 🔧 Troubleshooting

### Common Issues & Solutions

#### "Add to Home Screen" option not appearing
- **Solution**: Make sure you're using a supported browser (Chrome, Safari, Firefox)
- **Solution**: Wait for the page to fully load before trying to install
- **Solution**: Clear browser cache and try again

#### App not working offline
- **Solution**: Ensure you have a stable internet connection during first visit
- **Solution**: Try refreshing the page and reinstalling

#### App icon not appearing
- **Solution**: Restart your device after installation
- **Solution**: Check if the app is in your app drawer or home screen

#### Login issues
- **Solution**: Clear browser data and try again
- **Solution**: Ensure you're using the correct credentials
